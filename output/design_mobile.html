١ نظرة عامة

نظام ذكي يرصد الحوادث المرورية وتلف الطريق تلقائياً من خلال تحليل مقاطع فيديو الداش كام، باستخدام نموذج YOLOv8 المدرَّب على بيانات حقيقية. يشمل واجهة عربية تفاعلية مع خريطة جغرافية وتنبيهات فورية لجهات الاختصاص.

٥ سكريبتات المشروع
1_train.py
تحميل البيانات من Roboflow وتدريب YOLOv8s وقياس دقة النموذج
2_detect.py
تشغيل النموذج على فيديو الداش كام وإرسال تنبيه عند تأكيد الحادث
app.py
لوحة تحكم Streamlit كاملة بالعربي لكشف الحوادث وتلف الطريق
utils/map_utils.py
خريطة Folium تعرض موقع الحادث مع دائرة تنبيه بنصف قطر 1 كم
utils/road_damage_utils.py
كشف تلف الطريق (حفر وتشققات) وعرض النتائج بالعربي
utils/notify_utils.py
إرسال بريد إلكتروني للإسعاف والمرور يحتوي الوقت والموقع
config.py
ملف الإعدادات المركزي للمسارات والمعاملات وبيانات Roboflow

# ① تثبيت المكتبات pip install -r requirements.txt # ② تدريب النموذج (مرة واحدة) python3 1_train.py # ③ الكشف على فيديو python3 2_detect.py videos/accident_video.mp4 # ④ تشغيل لوحة التحكم streamlit run app.py
٧ الخلاصة

تطبيق عملي شامل لمفاهيم دورة رؤية الحاسب للمطورين المقدمة من سدايا، يربط بين جمع البيانات وتدريب النموذج وعرض النتائج في منظومة متكاملة، قابلة للتوسع نحو تطبيقات المدن الذكية والسلامة المرورية.
