## نظرة عامة

نظام ذكي يعمل على تحليل مقاطع فيديو كاميرات الداش كام للكشف التلقائي عن:
- **الحوادث المرورية** — باستخدام YOLOv8 المدرَّب على بيانات حقيقية
- **تلف الطريق** — حفر، تشققات طولية وعرضية وتمساحية

عند رصد حادث يقوم النظام بـ:
1. تمييز الحادث في الفيديو بصندوق تنبيه
2. عرض موقعه على خريطة تفاعلية
3. إرسال تنبيه بالبريد الإلكتروني للإسعاف والمرور

## المكتبات المستخدمة

| المكتبة | الدور |
|---------|-------|
| `ultralytics` | تدريب YOLOv8 وتشغيله |
| `opencv-python` | معالجة الفيديو والإطارات |
| `torch (PyTorch)` | محرك التعلم العميق |
| `roboflow` | تحميل Dataset التدريب |
| `streamlit` | واجهة لوحة التحكم |
| `folium` | الخريطة التفاعلية |
| `plotly` | الرسوم البيانية |
| `pillow` | رسم النصوص العربية |
| `arabic-reshaper + python-bidi` | دعم العربية في OpenCV |

## تشغيل المشروع

```bash
# ① تثبيت المكتبات
pip install -r requirements.txt

# ② تدريب النموذج — مرة واحدة فقط
python3 1_train.py

# ③ الكشف على فيديو
python3 2_detect.py videos/accident_video.mp4

# ④ تشغيل لوحة التحكم
streamlit run app.py
```

> **ملاحظة:** النماذج المدرَّبة موجودة في مجلد `models/` — لا حاجة لإعادة التدريب.
